Инструкция пользователя.

I Запуск приложения.
Для запуска приложения потребуется Intellij IDEA.
Шаги запуска программы:
1) Ввести в командную строку
git clone https://github.com/Sasakin/CalcApp.git
2) В Intellij IDEA
File -> New -> Project from existing sources  -> Create project from existing sources.
3) Добавить класс запускающий приложение
Edit Configurations -> Add New Configuration(Alt+Insert) ->
Application -> в поле Main class указать swingcalc.Main.
4) Запустить приложение

Если при совершении вышеперечисленных шагов запуск приложения неудался
рекомендуется проверить, чтобы пакет "com" имел маркировку "Sources Root", а пакет "resource" имел маркировку "Resources Root".
Для этого необходимо кликнуть правой кнопкой мыши на соответствующем пакете. Далее в пункте меню "Mark Directory as" выбрать соответствующую маркеровку пакета.

II Работа с приложением
После запуска приложения для полноценной работы потребуется подключить математические операции.
Это делается посредством выбора плагинов из левой части интерфейса приложения. Для ввода вещественных чисел предоставлен функционал состоящий из кнопок интерфейса приложения, а так же доступен ввод с клавиатуры. 
Вычисления в приложении "Калькулятор" происходит по следующему алгоритму:
1) Введите в текстовое поле вещественное число.
2) Выберите математическую операцию, которую хотите произмести.
3) Введите второе вещественное число.
4) Нажмите кнопку "=" , представленную в интерфейсе приложения.
5) После прохождения указанных выше шагов в текстовом поле приложения
отобразится результат вычисления.
