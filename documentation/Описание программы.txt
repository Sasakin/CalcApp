Описание программы.
Данная программа представляет собой десктопное приложение, реализующее функционал калькулятора. Приложение создано для совершения математических операций над вещественными числами. В функционал включены такие математические операции как сложение, вычитание, умножение, деление и извлечение квадратного корня. В приложении реализована идея добавления математической операции посредством выбора соответствующего плагина. Интерфейс приложения состоит из плагинов, включающих в приложение математические операции, а так же сетку кнопок и текстовое поле, необходимое для ввода вещественного числа. Для ввода вещественных чисел в приложении используется клавиатура либо кнопки из интерфейса приложения.
